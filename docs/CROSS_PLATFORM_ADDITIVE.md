# Cross-Platform & Additive-Only Improvements

## Overview

All tools now respect your work and are cross-platform compatible (Mac, iPad, Windows).

## Key Principles

### 1. **Additive Only - Never Overwrites**
- Tools only **add** missing elements
- Never replace or modify your existing content
- Only writes if content is **longer** (additive) or same length
- Checks if content already exists before adding

### 2. **Cross-Platform Compatibility**
- **Line Endings**: Preserves original line endings (Windows `\r\n`, Mac/iPad `\n`)
- **Path Handling**: Uses `path.join()` for all file paths (works on all platforms)
- **File Operations**: Uses Node.js `fs` module (cross-platform)

### 3. **Respects Your Scope & Creativity**
- Only adds missing attribution, license, ND joy comments
- Never modifies your class definitions or code structure
- Skips if content already present (case-insensitive check)
- Preserves your original formatting and style
