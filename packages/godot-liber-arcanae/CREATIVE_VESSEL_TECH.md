# Creative Vessel Tech - Game Integration

## Overview

Creative Vessel Tech is an alchemical transformation system for the game. Vessels are containers for transformation, like in real alchemy.

## Vessel Types

### 1. **Crucible** (Sulfur/Wands)
- **Purpose**: Intense transformation
- **Aesthetic**: Dynamic
- **Shader**: Cosmic energy streams
- **Use**: Fast, powerful transformations

### 2. **Alembic** (Mercury/Cups)
- **Purpose**: Distillation
- **Aesthetic**: Flowing
- **Shader**: Glowing orb
- **Use**: Purification, refinement

### 3. **Retort** (Salt/Swords)
- **Purpose**: Separation
- **Aesthetic**: Minimal
- **Shader**: Sacred geometry
- **Use**: Precision work

### 4. **Athanor** (Ash/Pentacles)
- **Purpose**: Slow, steady work
- **Aesthetic**: Harmonious
- **Shader**: Tree of Life
- **Use**: Long-term transformations

### 5. **Philosopher's Vessel** (All elements)
- **Purpose**: Complete transformation
- **Aesthetic**: Harmonious
- **Shader**: Duality
- **Use**: Perfect transformations

### 6. **Creative Vessel** (Aether)
- **Purpose**: Creative neurodivergent process
- **Aesthetic**: Flowing
- **Shader**: Cosmic sky
- **Use**: Creative manifestations

## Integration

### Codex → Vessels
- Each codex node can create a vessel
- Vessel correspondences from codex
- Flow calculations from codex aesthetic

### Characters → Vessels
- Characters can own vessels
- Vessel properties from character data
- Creative flow from character playstyle

### Creative Process → Vessels
- Vessels respond to creative states
- Manifestations based on creative flow
- Shader effects from creative process

## Shader Effects

All vessels have beautiful shaders:
- **Cosmic energy streams** (Crucible)
- **Glowing orbs** (Alembic)
- **Sacred geometry** (Retort)
- **Tree of Life** (Athanor)
- **Duality** (Philosopher)
- **Cosmic sky** (Creative)

## Game Mechanics

1. **Fill Vessel**: Add energy to vessel
2. **Transform**: Vessel transforms energy over time
3. **Manifest**: Vessel creates result
4. **Use Result**: Apply manifestation to game

## High-End Japanese Tech Aesthetic

- **Minimal**: Clean, simple interfaces
- **Flowing**: Smooth animations
- **Harmonious**: Balanced, beautiful
- **Dynamic**: Responsive, alive

**Everything flows together beautifully - like high-end Japanese tech with mystical alchemical art.**
