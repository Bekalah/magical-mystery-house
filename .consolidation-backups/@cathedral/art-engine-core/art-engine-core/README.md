# @cathedral/art-engine-core

**Sacred Geometry & Visual Creation Engine**

Creates art from Codex144 nodes using sacred geometry, integrating 144:99 ratio, golden ratio, Fibonacci, and consciousness mapping.

## Features

- Sacred geometry shapes and patterns
- Consciousness-based color palettes (0-21 levels)
- Pattern generation with sacred ratios
- Composition layouts (golden ratio, 144:99, Fibonacci)
- SVG generation
- Museum-grade quality standards

## Usage

```typescript
import { ArtEngine } from '@cathedral/art-engine-core';

const engine = new ArtEngine();
const artNode = engine.createArtNode(42);
const svg = engine.generateSVG(artNode);
```
